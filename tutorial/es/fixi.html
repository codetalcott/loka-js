fixi — atributos HTML para HTTP y reemplazo de HTML

fixi.js añade seis atributos HTML que permiten enviar una petición HTTP cuando ocurre un evento y reemplazar parte de la página con la respuesta — sin escribir JavaScript.

En español, los atributos se escriben en español: fx-acción, fx-objetivo, fx-disparador, fx-método, fx-intercambio. El navegador y el devtools los conservan tal cual — fixi los resuelve al momento de leerlos.

Ejemplo: cargar un fragmento

Español

— vacío —
<button fx-acción="hola.html"
        fx-objetivo="#salida"
        fx-intercambio="innerHTML">
  Cargar saludo
</button>
<div id="salida"></div>

Canonical (en)

— empty —
<button fx-action="hello.html"
        fx-target="#out"
        fx-swap="innerHTML">
  Load greeting
</button>
<div id="out"></div>
