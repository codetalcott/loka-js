rexi — envoltorio cómodo de fetch() para llamadas desde JS

rexi.js expone seis verbos HTTP como funciones globales (get, head, post, put, patch, del) que devuelven una promesa decorada: encadenas .text(), .json(), .html() o .blob() sin la danza del doble await.

En español, los mismos verbos están disponibles bajo: obtener, cabecera, publicar, poner, parchear, eliminar. Apuntan al mismo objeto función; ambos nombres siguen funcionando.

Ejemplo: cargar texto

Español

— vacío —
let texto = await obtener("./hello.txt").text()
document.getElementById('out').textContent = texto

Canonical (en)

— empty —
let text = await get("./hello.txt").text()
document.getElementById('out').textContent = text

Vocabulario

EspañolInglés (canónico)Verbo HTTP
obtener(url)get(url)GET
cabecera(url)head(url)HEAD
publicar(url, body)post(url, body)POST
poner(url, body)put(url, body)PUT
parchear(url, body)patch(url, body)PATCH
eliminar(url)del(url)DELETE

Los métodos encadenados (.json(), .text(), .html(), .blob()) permanecen en inglés — son nombres establecidos del SDK.
