Triage de incidente #42 — "Caída de base de datos en eu-west"

Severidad: sin clasificar  |  Responsable: sin asignar  |  Estado: alerta recibida

Este es un manifiesto de afordances en español para el dominio incident_response de grail-domains. Un agente LLM (o el script en ../incident-triage-es/agent.mjs) lee esta página, traduce los términos al inglés canónico via la tabla conditions del locale es, y ejecuta los pasos contra el backend en http://127.0.0.1:8091.

1. Reconocer incidente

El primer paso del ciclo de vida.
→ incidente.reconocido

2. Clasificar severidad

Asignar P1–P4 al incidente.
← incidente.reconocido → incidente.severidad.clasificada

3. Asignar responsable

Persona responsable del incidente.
← incidente.severidad.clasificada → incidente.responsable.asignado

4. Investigar causa raíz

Documentar los hallazgos.
← incidente.responsable.asignado → incidente.investigado

5. Mitigar

Aplicar la mitigación.
← incidente.investigado → incidente.mitigado

6. Verificar salud del sistema

Confirmar que los chequeos pasan post-mitigación.
← incidente.mitigado → incidente.salud.verificada

7. Notificar interesados

Comunicación post-mitigación.
← incidente.mitigado → incidente.interesados.notificados

8. Escribir postmortem

Documento de análisis post-incidente.
← incidente.salud.verificada + incidente.interesados.notificados → incidente.postmortem.escrito

9. Crear acciones de seguimiento

Items de acción derivados del postmortem.
← incidente.postmortem.escrito → incidente.acciones.creadas

10. Resolver incidente

Cierre formal del ciclo de vida.
← incidente.acciones.creadas → incidente.resuelto
Cómo correr este demo:
  1. Iniciar el backend GRAIL: cd ~/projects/grail-domains && .venv/bin/python -m domains.incident_response.server (puerto 8091)
  2. Servir esta página: npx http-server . -p 3036 -s desde la raíz del repo loka-js
  3. Leer las afordances: node tools/agent-reader.mjs http://127.0.0.1:3036/examples/incident-triage-es/index.html --validate
  4. Correr el agente automático: node examples/incident-triage-es/agent.mjs
