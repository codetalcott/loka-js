fixi + psatina, juntos en español

Una sola página combina dos bibliotecas multilingües. Sin mutar el DOM en ningún lado. Abre DevTools → Elementos para confirmar que los nombres localizados (fx-acción, p:datos, p:en:entrada) permanecen en la fuente HTML.

Vocabulario compartido: loka-js/dom-vocab/es.js exporta { events, props } que psatina-modular importa. Una sola fuente de verdad para clic→click, entrada→input, valor→value, etc.

psatina-modular — isla reactiva (Spanish-native)

Directivas: p:datos, p:para, p:en:entrada, p:si. Reactividad pura del cliente, sin red.

fixi — intercambio de fragmento (Spanish-native)

Atributos: fx-acción, fx-objetivo, fx-intercambio. El fragmento descargado contiene su propia plantilla psatina, que el helper rescan.js inicializa al recibir el evento fx:swapped.
