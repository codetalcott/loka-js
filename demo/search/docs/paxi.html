paxi.js

paxi.js (0.6 KB Brotli) is a tiny morphing library that swaps HTML into the DOM by reconciling the old tree with the new one rather than replacing wholesale. This preserves focus, caret position, scroll offset, and form data during updates.

API

paxi adds no new attributes of its own. Integration points are:

  • fx-swap="morph" on a fixi-powered element to enable morphing for that swap.
  • window.morph(target, html) for standalone use outside fixi — works with HTML from any source: local state, JSON APIs, WebSocket messages.

Loaded alongside fixi, paxi automatically registers as the morph swap strategy. You can set morphing as the default app-wide via fixiCfg.

Example

Reload a profile section while preserving an in-progress nickname input:

<button fx-action="/profile" fx-target="#profile" fx-swap="morph">
  Reload profile
</button>
<form id="profile">
  <input id="nickname" placeholder="text survives the swap">
</form>

ID-keyed reconciliation preserves the input's value, focus, and selection across the swap.
