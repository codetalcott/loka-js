Demos

Six example applications demonstrate the libraries in real use:

  1. Contacts CRUD — fixi + paxi + moxi, with active search and inline editing.
  2. TodoMVC clone — same stack, classic to-do list interface.
  3. Streaming AI bot — fixi + ssexi + moxi, server-pushed pseudo-AI replies.
  4. Live chat — fixi + ssexi + moxi, real shared chat over an SSE broadcast.
  5. Dashboard — fixi + ssexi, fanned-out live dashboard pattern.
  6. Snake game — all four (fixi + ssexi + rexi + moxi), interactive gameplay.

Run them locally

git clone https://github.com/bigskysoftware/the-fixi-project.git
cd the-fixi-project
npm install
npm run demo

The demo server runs at http://localhost:8765.

Together these examples illustrate patterns for real-time interactivity, server-sent events, DOM morphing, and ergonomic fetch operations across practical use cases.
