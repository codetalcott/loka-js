moxi.js

moxi.js is the client-side scripting component of the fixi project — a small companion that lets you write inline JavaScript handlers and DOM-reactive expressions without ceremony.

Attributes

  • on-<event> — runs JavaScript when the named event fires on the element.
  • live — re-evaluates JavaScript when DOM or form state changes.

Modifiers

Handlers chain modifiers via dots: on-click.prevent.stop="…"

  • .prevent / .stop / .halt — control event propagation
  • .once — fires only on first occurrence
  • .self — skip events bubbled from descendants
  • .outside — fires only for events outside the element
  • .capture / .passive — addEventListener options
  • .cc — convert event name to camelCase (web components)

Globals

  • q(x) — query selector returning a proxy with collection methods
  • wait(x) — promise resolving after milliseconds or a named event
  • transition(fn) — wraps in the View Transition API
  • trigger / debounce — handler-scoped helpers

Example

<input id="name" placeholder="type your name">
<output live="this.innerText = q('#name').value
  ? 'hello ' + q('#name').value : ''"></output>
<button on-click="q('#name').value = ''">clear</button>
