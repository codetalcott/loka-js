fixi.js

fixi.js (3.4 KB raw / 1.2 KB Brotli) is a miniature take on htmx. It enables HTML elements to trigger HTTP requests on user events and insert the response anywhere in the document — declaratively, with no scripting required.

Attributes

AttributePurposeDefault
fx-actionRequest URLrequired
fx-methodHTTP verbGET
fx-triggerDOM event that triggers the requestelement-dependent
fx-targetCSS selector for the swap targetthe element itself
fx-swapInsertion mechanismouterHTML

Lifecycle events

  1. fx:init — before element setup (cancelable)
  2. fx:inited — after wiring complete
  3. fx:config — request triggered, before sending (mutate cfg)
  4. fx:before — immediately before fetch()
  5. fx:after — response received, before swap
  6. fx:swapped — content inserted, transitions complete
  7. fx:error — fetch() exception
  8. fx:finally — after every request

Example

<button fx-action="/profile"
        fx-target="#panel"
        fx-swap="innerHTML">
  Load profile
</button>
<output id="panel"></output>

Clicking triggers a GET to /profile; the response replaces the contents of #panel.
