The Fixi Project

A collection of small web libraries offering lightweight takes on established tools — inspired by HTMX, Hyperscript, and Idiomorph. Each library is sharply focused, and they're designed to compose.

LibraryBrotliPurpose
fixi.js1.2 KBSupercharged HTML — element-driven HTTP requests + DOM swaps
moxi.js1.8 KBInline scripting + DOM-based reactivity
paxi.js0.6 KBDOM morphing that preserves focus and form state
ssexi.js1.4 KBStreaming HTML over Server-Sent Events
rexi.js1.4 KBErgonomic fetch() wrappers

Install

The all-in-one bundle is 4.5 KB compressed:

<script src="https://cdn.jsdelivr.net/npm/the-fixi-project@0.1.1/dist/the-fixi-project.min.js"></script>

Or via npm:

npm install the-fixi-project
